Opticon

Financial terminal with live markets, situation monitor, personal finance, and a paper-trading simulator

20+
API Routes

Situation Monitor

  • Full-page live map with tactical HUD overlay
  • Traffic incidents and construction barriers (OSM)
  • Real-time earthquake markers (USGS)
  • Crime data feeds with local overlays
  • Local events: concerts, sports, community (PredictHQ)
  • Global geopolitical events (GDELT)
  • Viewport-based refresh: pan to a new city, overlays update

Markets

  • Scrolling ticker: Mag 7, 100 US stocks, commodities, ETFs
  • FMP batch quotes (primary) + Yahoo Finance (fallback)
  • Prediction markets: Polymarket feed with geo-anchored map pins
  • Paper-trading simulator: $1 to $1T with Fibonacci checkpoints

Personal Finance

  • Portfolio panel: live-valued holdings, accounts, net worth
  • Budget tracker: income, expenses, surplus
  • Debt tracker with payoff timeline
  • Spending trends with category breakdown
  • Server-side persistence via /api/portfolio (Vercel KV)
  • CLI access: balance command for terminal summary

API

  • /api/stocks-free -- batch stock quotes (FMP/Yahoo)
  • /api/portfolio -- portfolio CRUD (get, update, summary)
  • /api/auth -- register, login, logout, verify-email
  • /api/traffic -- traffic incidents
  • /api/earthquakes -- USGS seismic feed
  • /api/events -- GDELT geopolitical events
  • /api/crime -- crime data
  • /api/weather -- weather + alerts
  • /api/flights -- flight tracking

Quick Start

git clone https://github.com/nulljosh/opticon.git
cd opticon
npm install
npm run dev

Stack

  • React 19 + Vite
  • MapLibre GL + CartoCDN tiles
  • Vercel serverless (single gateway function)
  • Vercel KV (Upstash Redis) for auth + portfolio
  • FMP + Yahoo Finance + Polymarket + GDELT + USGS + OSM
  • Vitest + Playwright
